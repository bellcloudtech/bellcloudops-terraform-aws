\# BellCloudOps Terraform AWS Infrastructure Lab



\## Overview



This project demonstrates how to deploy a cloud infrastructure environment in AWS using Terraform.



The environment provisions a custom VPC, networking resources, and an EC2 web server that hosts a demo website for the fictional cloud company \*\*BellCloudOps\*\*.



\## Architecture



The Terraform configuration creates the following AWS resources:



\- VPC

\- Public Subnet

\- Private Subnet

\- Internet Gateway

\- Route Table

\- Route Table Association

\- Security Group

\- EC2 Web Server running Apache



The EC2 instance serves a simple website demonstrating a fictional cloud services company.



\## Technologies Used



\- Terraform

\- AWS EC2

\- AWS VPC

\- AWS Networking (Subnets, Route Tables, Internet Gateway)

\- Apache HTTP Server
